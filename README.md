# KISA 핀테크 아카데미 수강

참가 날짜 : 2020.02.22 ~ 2020.02.26 
프로젝트 인원 : 개인

### 1) 프로젝트 목표
금융결제원에서 제공하는 오픈 API를 이용한 간편 결제 시스템 구현 

### 2) 프로젝트 기술스택

BackEnd : node.js </br>
DataBase : Mysql</br>
FrontEnd : 외부 라이브러리 사용</br>
오픈 API : https://developers.kftc.or.kr/dev </br>
배포 : Heroku</br>
배포 DB : ClearDB</br>
사용자 인증 : JWT Token</br>
API 테스팅 : Postman

### 3) 프로젝트 시스템 설계도

사용 API : 잔액조회, 거래내역조회, 계좌실명조회, 입금이체, 출금이체
